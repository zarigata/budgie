Simple, Distributed Container Orchestration

Budgie is a lightweight tool for running, discovering, and replicating containers on your local network. No complex setup, just simple `.bun` files.

Get Started View on GitHub

Features

LAN Discovery

Automatically discover containers on your local network using mDNS. No manual IP configuration needed.

Simple Config

Define containers with readable YAML `.bun` files. Ports, volumes, health checks‚Äîall in one place.

Data Sync

Automatically synchronize volumes across replicas using an efficient rsync-style algorithm.

Load Balancing

Built-in HTTP reverse proxy with round-robin and least-connections algorithms. Health checks included.

Quick Start

1. Install Budgie

Linux / macOS:

curl -fsSL https://zarigata.github.io/budgie/install.sh | sudo bash

Windows (PowerShell as Admin):

irm https://zarigata.github.io/budgie/install.ps1 | iex

2. Create a .bun file


# webapp.bun
version: "1.0"
name: "webapp"

image:
  docker_image: "nginx:alpine"

ports:
  - container_port: 80
    host_port: 8080

3. Run it

budgie run webapp.bun

4. Discover it

budgie chirp
